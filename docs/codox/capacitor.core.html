make-client
(make-client opts)
Returns a map representing an HTTP client configuration.

Valid options:
  :host         (default: "localhost")
  :scheme       (default: "http://")
  :port         (default: 8086)
  :username     (default "root")
  :password     (default "root")
  :db           (default: "testdb")
  :version      (default: "0.9")
  :post-opts    (http post options, default: "nil")
  :get-opts     (http get options, default: "nil")
